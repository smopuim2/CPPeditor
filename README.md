## What is CPPeditor ?
CPPeditor is an IDE of c++ .
You can edit and run simple code on it .

## How to use CPPeditor ?
Edit file: just enter your code .
Other functions' hotkeys:
1. `^F` -- View files .
2. `^O` -- Open a file .
3. `^S` -- Save this file .
4. `^R` -- Run a file .
5. `^Q` -- Open the FAQs/help .
6. `^X` -- Exit .
