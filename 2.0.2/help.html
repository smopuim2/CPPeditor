What is CPPeditor ?

CPPeditor is an IDE of c++ .
You can edit and run simple code on it . (It's size is smaller than 2MB !)

How to use CPPeditor ?

Edit file: just enter your code .
Other functions' hotkeys: 

  • ^F
    -- View files .
  • ^O
    -- Open a file .
  • ^S
    -- Save this file .
  • ^R
    -- Run a file .
  • ^Q
    -- Open the FAQs/help .
  • ^X
    -- Exit .

You can reset your editor's color option .
Open the CPPeditor's root directory, then edit 
show.ini
:
; style ini file
[color]
bgclr=0  //background color
fgclr=15 //foreground color
hilt=3   //highlight color(fg)
csr=112  //cursor color(bg+fg)
bttm=6   //the bottom title's color(fg)
[size]
line=17  //max lines showing
colu=70  //max columns showing

And the number of color is show below: (in 
<wincon.h>
)
#define FOREGROUND_BLUE 0x1
#define FOREGROUND_GREEN 0x2
#define FOREGROUND_RED 0x4
#define FOREGROUND_INTENSITY 0x8
#define BACKGROUND_BLUE 0x10
#define BACKGROUND_GREEN 0x20
#define BACKGROUND_RED 0x40
#define BACKGROUND_INTENSITY 0x80

Use operation BINARY_OR to combine them .
